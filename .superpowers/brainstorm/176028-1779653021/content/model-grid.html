The OLOS 6-Box Model (as written)

3 stages × 2 phases. The doc claims a user is only ever in one box at a time.

Initiation — "set up the baseline"
Operation — "keep it alive"
OBSERVE
What is true?

Baseline reality

Hydrology, soil, topo, climate, access, SWOT → Baseline Land Intelligence Package

Ongoing monitoring

Water after rain, erosion, species, hazards → Updated obs + Plan-impact flags

PLAN
What should we do?

Development plan

Land use, water, roads, paddocks, phasing, budget → Approved Development Plan

Keep plan current

Revise from new obs, generate tasks, re-sequence → Operating Plan + task packages

ACT
What work happens?

Build the project

Earthworks, fencing, planting, water systems → Completed dev work + proof

Run & maintain

Rotate stock, prune, repair, seasonal routines → Completed ops + maintenance records

Initiation = one-time setup (thin slice at the front) Operation = forever (where the property actually lives)
