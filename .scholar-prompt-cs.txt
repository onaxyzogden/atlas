PROJECT: Atlas — Plan stage, module: Cross-section & Solar Geometry (Module 6)

OPTION A — Atlas current (TransectVerticalEditorCard.tsx, ~540L, full store wiring):
Sophisticated vertical-section editor. Steward picks a saved transect (line drawn on the live map between two points). Along that transect they pin vertical elements at metric distances; each pin is either a *standalone* element (type=structure|tree|shrub|swale|pond + height m + label) or a *typed reference* into another store (water earthworks, polyculture guilds & species, closed-loop fertility units, structures) — the linked ref's height + label resolves at render time from defaults (e.g. swale 0.5m, cistern 2.5m, pond 1.0m, hugelkultur 1.2m). Renders a 2D profile view with element silhouettes. Includes solar overlay: latitude is read from transect.pointA[1], and winter / summer solstice noon altitude lines are drawn so the steward can see whether trees/structures shade solar collectors.
