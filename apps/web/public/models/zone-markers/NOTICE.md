# `models/zone-markers/` — attribution

Symbolic GLB markers for Permaculture Zones 0–5 consumed by the
Plan-stage `DesignElementScenegraphLayer` (deck.gl).

All meshes in this directory are CC0-licensed code-art generated by
`apps/web/scripts/gen-zone-marker-glbs.mjs`. Each marker is the same
pillar geometry; only the material colour distinguishes the zone.
