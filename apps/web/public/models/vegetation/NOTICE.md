# `models/vegetation/` — attribution

Per-kind GLB models for vegetation (trees and shrubs) consumed by the
Plan-stage `DesignElementScenegraphLayer` (deck.gl). Hedgerow is line
geometry and renders via the 2D fill in `BeV2GenericLayer`, so it has
no GLB here.

All meshes in this directory are CC0-licensed code-art generated by
`apps/web/scripts/gen-vegetation-glbs.mjs` — not lifted from external
asset packs. Re-run `pnpm --filter @ogden/web gen:models` to regenerate.

| `apple-tree.glb` | Atlas build script | `apps/web/scripts/gen-vegetation-glbs.mjs` (`apple-tree`) | CC0-1.0 | Thin trunk + medium rounded dome canopy, lighter green. |

## Adding new vegetation models

1. Add a kind to `apps/web/scripts/gen-vegetation-glbs.mjs` (use the
   existing `deciduousTree`, `coniferTree`, or any `primitives.mjs`
   builder).
2. Run `pnpm --filter @ogden/web gen:models`.
3. Append a row above with author, source, licence, notes.
4. Register the kind in
   - `packages/shared/src/builtEnvironmentKinds.ts` (category `vegetation`)
   - `apps/web/src/v3/plan/canvas/elementCatalog.ts` (category `vegetation`)
   - `apps/web/src/v3/plan/canvas/elementHeights.ts` (so it participates in 3D)
