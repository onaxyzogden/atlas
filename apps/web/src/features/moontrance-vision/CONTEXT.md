## Phase tags
P2, P3, MT (see `packages/shared/src/featureManifest.ts` for the authoritative item list)

## Purpose
Hold the *vision* of the land alongside its *current* state: toggle between
as-is terrain and the concept masterplan, reveal the design stage-by-stage,
and frame features in the language of purpose — "why this is here", "what
this replaces", "what problem this solves". MT-tagged items layer the
Moontrance voice on top: signs-in-creation interpretation, spiritual and
educational storytelling, service and stewardship framing. P3 items add
guided flyover, curated visitor journeys, and vision-board media.

## Known gotchas
- **MT phase gate.** Four items are phase `MT`:
  `signs-in-creation-interpretive-mode`, `spiritual-educational-storytelling-layer`,
  `service-stewardship-framing-panels`, `feature-explanations-tied-to-purpose`.
  They are hidden/404'd unless the `ATLAS_MOONTRANCE=1` flag is set.
  `phaseAtMost()` does not include MT in P-tag ordinal comparisons —
  gate MT separately.
- **Legacy folder overlap.** `features/vision/` and `features/moontrance/`
  both predate §14 and already implement the overlay toggle, Moontrance
  identity fields, and phase-filter plumbing. New §14 work MUST route
  through the existing stores (`useVisionStore`, `useMapStore`) instead of
  standing up a parallel slice. See the Redirect notice.
- **Phase filter semantics.** `setActivePhaseFilter('all')` reveals the full
  concept overlay; per-phase values ('Phase 1', etc.) drive the
  stage-by-stage reveal. The toggle in `VisionPanel` currently flips
  between 'all' and 'Phase 1' — stage-by-stage will need a richer control.
- **Moontrance tab visibility.** The Moontrance tab in `VisionPanel` only
  shows when `project.projectType === 'moontrance'`. Generic P2 narrative
  mode must not leak MT copy for non-Moontrance projects.
- **P3 flyover / camera angles** imply Cesium camera scripting. Ensure the
  map is in 2.5D/3D mode (Section 2) before exposing the control.
- **Vision board media (P3)** needs a storage path for reference imagery —
  no provider selected yet; scope before building UI.
