## Phase tags
P1, P2, MT (see `packages/shared/src/featureManifest.ts` for the authoritative item list)

## Purpose
Owns the design-time model of how a property is carved into functional
zones: drawing custom polygons, naming and color-coding them, assigning
primary/secondary use categories (habitation, food production, livestock,
commons, spiritual, education, retreat, conservation, water retention,
infrastructure, access, buffer, future expansion), running sizing and
allocation math against total acreage, and surfacing overlap/conflict
/adjacency analysis plus site-analysis-driven auto-suggestions. MT-phase
items extend this with prayer/spiritual-zone, quiet-contemplation,
women/family privacy, and men's-cohort zone planning that only activate
under the Moontrance flag.

## Redirect notice

## Known gotchas
- Four features in this section are MT-tagged (prayer/spiritual,
  quiet contemplation, women/family privacy, men's cohort). Routes
  and UI surfaces for these must gate on `fastify.requirePhase('MT')`
  server-side and the matching `ATLAS_MOONTRANCE=1` flag client-side.
  `phaseAtMost('MT', 'P4')` returns `false` — MT does not inherit
  from P-ladder caps. Test both a `ATLAS_PHASE_MAX=P4` build (MT
  items hidden) and a moontrance-enabled build (MT items visible).
- `ZONE_CATEGORY_CONFIG` in `store/zoneStore.ts` does not currently
  split primary vs secondary use. The "primary/secondary use
  designation" P1 feature needs a new field on `LandZone` when
  promoted from `partial` to `done`.
- Event, parking, seasonal, temporary, and phased-use zones are not
  yet represented as zone categories. Add them to the `ZoneCategory`
  union and `ZONE_CATEGORY_CONFIG` before building UI.
- `zoneAnalysis.ts` already implements sizing, conflict detection,
  allocation, and auto-suggest. Re-use those functions from
  `ZoningAllocationPage` rather than re-implementing.
- `features/spiritual/QuietZonePlanning.tsx` reads `LandZone[]`
  directly — any change to zone categories (adding `quiet` as its
  own category, etc.) must propagate there.
