## Purpose
Synthesizes every collected data layer (terrain, hydrology, soil, climate,
zoning, regulatory) into a single diagnostic dossier for a property:
scored labels with confidence indicators, detected site flags (steep
slope, flood-prone, frost pocket, wind exposure, cold-air drainage,
microclimate zones), candidate intervention zones (pond/swale/keyline),
restoration priorities, and AI-generated narrative summaries ("what this
land wants", threats, highest-leverage interventions). This is the
output of the data pipeline and the input to every downstream design
section.
