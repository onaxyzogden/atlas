## Phase tags

## Purpose
Section 19 turns the atlas into a teachable surface. Clickable hotspots and
side-panel explanations answer "why is this feature here, what problem does
it solve, what happens if omitted?" for every placed element. In the MT
timeline, those explanations extend into rationale cards tied to purpose and
meaning, explanation modes per discipline (ecology, water, livestock,
agroforestry, spiritual symbolism, regeneration phase), and before/after
storytelling panels. P4 widens the audience: passive tours, voiceover-ready
script export, slide presentation mode, public/stakeholder modes, operator
training, and student quiz/checkpoint mode.
